What is Sass and LESS

- They're both preprocessors that are used to keep CSS more maintainable, themable, and extendable by allowing them to use variables, mixins and functions.

- A CSS preprocessor is a scripting language that extends and then compilesit into regular CSS.

More Similarities Between the Two

- Both are CSS extenders that are backwards compatible, making it easy to convert existing CSS files.

- Both need to be compiled because browsers cannot translate or interpret either themselves.

Differences between LESS and Sass

Sass uses Ruby.

LESS uses Javascript.

Sass uses $, Less use @.

Less is more visually appealing and easier to follow for first time users.

Installing LESS

You need to download NodeJS installer. For Mac or Linux you just use the commands

1. Open the command prompt.

2. Run npm install less

Use this command to compile to CSS.lessc styles.less styles.css

Installing Sass

If you're not using a Mac, then you have to download a Ruby installer.

1. Open Command prompt.

2. Run gem install sass

Use this command to to compile CSS.sass --watch styles.scss
